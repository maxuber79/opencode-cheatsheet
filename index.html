Bonus · Comandos personalizados

La capa más potente de OpenCode: convierte prompts repetitivos en comandos propios. Crea un archivo .opencode/commands/componente.md con tus convenciones ya incorporadas y llámalo con /componente.

---
description: Genera un componente Angular con las convenciones del proyecto
agent: build
model: anthropic/claude-sonnet-4-6
---
Crea un componente Angular llamado $ARGUMENTS.
Aplica SIEMPRE estas reglas:
1. JSDoc en cada función.
2. const DEBUG = false; al inicio del archivo.
3. console.log solo dentro de bloques if (DEBUG).
4. Trabaja paso a paso, un archivo por vez.
5. Termina con un resumen y una pregunta antes de continuar.

Comandos · Crear estructura al instante

Copia y ejecuta uno de estos comandos para crear toda la estructura AI-Ready en tu proyecto.

# Git Bash / Linux / macOS
mkdir -p docs && touch AI.md README.md docs/{ARCHITECTURE,STYLEGUIDE,DECISIONS,TODO,CHANGELOG,CONTRIBUTING,SECURITY,PROGRESS,PROMPTS}.md
# PowerShell
New-Item -ItemType Directory -Path docs -Force
'AI.md','README.md' | % { New-Item $_ -Force }
'ARCHITECTURE','STYLEGUIDE','DECISIONS','TODO',
'CHANGELOG','CONTRIBUTING','SECURITY','PROGRESS',
'PROMPTS' | % { New-Item "docs/$_.md" -Force }
